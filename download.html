Download Ethereal Aion 3.9

The Ethereal Aion Launcher is all you need to begin your adventure!

Prerequisite: To run the launcher, you need to have .NET Desktop Runtime 6.0.4 (x86) installed.

Ethereal Aion Launcher

Size: 3MB

OR

Download Client

Size: 15.7GB (20GB extracted)

Installation Instructions:

1. Download both compressed client parts, place them in a dedicated folder then extract them.

2. Download the launcher, then place it inside your game client.

3. Run the launcher, let it update your client and get ready to play!

(It is also possible to let the launcher download the whole game inside of an empty folder.)

Quick Troubleshooting:

- If your antivirus does not let you download it or keeps killing it, consider disabling it before downloading and/or add EtherealAion.exe as an exception.

- If the launcher suddenly shuts down, please check the Launcher_logs.txt file.
